# Figma Design Specification - Crispy Express

## Pages
1. **Design System**: Color palette, typography, spacing, icons.
2. **Desktop (1440px)**: Full landing page, menu grid, cart drawer.
3. **Tablet (768px)**: Responsive layout adjustments.
4. **Mobile (375px)**: Mobile-first layout, bottom CTA bar.

## Design Tokens
- **Colors**:
  - Primary: #c20a0a (Bold Red)
  - Accent: #f6c24b (Gold)
  - Charcoal: #111111 (Deep Black)
  - White: #ffffff
- **Typography**:
  - Headings: Montserrat (Black/Bold)
  - Body: Inter (Regular/Medium)
- **Spacing Scale**: 4, 8, 16, 24, 32, 48, 64px.
- **Border Radius**: 12px (Small), 24px (Large), 9999px (Pill).

## Components
- **Nav**: Sticky, logo left, links center, icons right.
- **Hero**: Centered text, background image with overlay.
- **Product Card**: Image top, content bottom, primary CTA button.
- **Cart Drawer**: Right-side slide-in, item list, subtotal, checkout button.
- **Badges**: "Best Seller", "New", "Deal".
- **Buttons**:
  - Primary: Solid red, white text, rounded.
  - Secondary: White background, red border, red text.

## Assets
- **Photography**: 
  - Hero: 1920x1080 (High contrast food photography)
  - Products: 600x400 (Close-up appetizing shots)
- **Icons**: Lucide React set (ShoppingBag, Menu, X, Star, Clock, Truck).
