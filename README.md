# Crispy Express - High Conversion Fast Food Platform

This is a production-ready fast-food ordering website built with React, Vite, and Express.

## Tech Stack
- **Frontend**: React 19, Tailwind CSS 4, Framer Motion, Lucide React
- **Backend**: Express (Node.js)
- **State**: React Context (Cart)
- **SEO**: React Helmet Async + JSON-LD
- **Payments**: Stripe (Integration Placeholder)

## Features
- Mobile-first responsive design
- Real-time cart management with local persistence
- High-conversion landing page with social proof
- Category-based menu filtering
- Multi-step checkout flow
- SEO optimized with structured data

## Setup & Deployment
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`
4. Start production server: `npm start`

## Environment Variables
- `GEMINI_API_KEY`: For AI features
- `STRIPE_SECRET_KEY`: For payment processing
- `APP_URL`: Base URL of the application
